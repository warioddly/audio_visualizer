Audio Visualizer

Web audio visualizer based on the Three.js library. And the source code is written in TypeScript

WebGL, Three.js, TypeScript, Audio Visualizer, Web Audio API


Select an audio file to play

Press Space To play/pause

Press Arrow right To fast-forward

Press Arrow right To rewind

Press Arrow up To increase volume

Press Arrow down To decrease volume
